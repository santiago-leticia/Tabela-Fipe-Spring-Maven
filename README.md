<h1>Java: trabalhando com lambdas, streams e Spring Framework</h1>
##

Dentro da atividade do curso Java: trabalhando com lambdas, streams e Spring Framework, tiver praticar a utilização de listas, coleções de dados, funções lambdas, streams, geração de estatísticas e vários outros recursos do Java. Você criou um projeto Spring com Maven implementando a interface CommandLineRunner para executar a aplicação. Consumiu a API do OMDb, recuperando dados de séries, temporadas e episódios. Fez a modelagem de dados e implementou a desserialização do JSON utilizando a biblioteca Jackson, fazendo proveito das anotações como @JsonAlias e @JsonProperties para personalizar as informações para o seu contexto. Aprendi também criar interfaces e classes de serviço, criando método com retorno genérico para evitar redundância de código. Criei um menu de opções, utilizou listas com operações diversas, como filtro, ordenação, transformação de dados, geração de novas listas e mapas. Fez buscar a partir de um nome de episódio ou a partir de uma data específica, gerou estatísticas de avaliação por temporada e de toda a série de forma geral, identificando o episódio com maior e menor avaliação, conferindo de fato a média das validações e ainda viu como conectar o seu repositório ao GitHub, para praticar o trabalho colaborativo.
